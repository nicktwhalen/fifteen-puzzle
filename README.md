# 15 Puzzle

A simple 15 puzzle game.

## Getting Started

Install dependencies:

```bash
yarn install
```

Run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Development

```bash
# Run tests
yarn test

# Type checking
yarn type-check

# Lint TypeScript
yarn lint

# Lint CSS
yarn lint:css

# Run all checks
yarn lint:all

# Build for production
yarn build
```
